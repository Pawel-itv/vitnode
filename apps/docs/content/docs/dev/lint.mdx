Lint helps you to keep your code clean and consistent. VitNode uses [ESLint](https://eslint.org/) and [Prettier](https://prettier.io/) to lint the code.

## Checking Code

import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"
pnpm lint
```

```bash tab="npm"
npm run lint
```

</Tabs>

## Fixing Code

<Tabs id='package-manager' persist items={['pnpm', 'npm']}>

```bash tab="pnpm"
pnpm lint:fix
```

```bash tab="npm"
npm run lint:fix
```

</Tabs>
