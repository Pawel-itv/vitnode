# Captcha

VitNode provides a simple way to integrate captcha in your application to prevent bots from submitting forms. It supports multiple captcha providers and can be easily integrated into your plugin.

VitNode has support for:

- [Google reCAPTCHA v2 (checkbox)](https://www.google.com/recaptcha/about/)
- [Google reCAPTCHA v2 (invisible)](https://www.google.com/recaptcha/about/)
- [Google reCAPTCHA v3](https://www.google.com/recaptcha/about/)
- [Cloudflare Turnstile](https://www.cloudflare.com/products/turnstile/)

## Usage

<Steps>
### Backend - Service

Inside your service file, import the `CaptchaCoreCaptchaSecurityService` and inject it into your service.

```ts {5, 10, 14}
import { Injectable } from '@nestjs/common';

import { SignUpCoreSessionsObj } from './dto/sign_up.obj';

import { Ctx, CaptchaCoreCaptchaSecurityService } from 'vitnode-backend';

@Injectable()
export class SignUpCoreSessionsService {
  constructor(
    private readonly captchaService: CaptchaCoreCaptchaSecurityService,
  ) {}

  async signUp({ req }: Ctx): Promise<SignUpCoreSessionsObj> {
    await this.captchaService.validateCaptcha({ req });
  }
}
```

### Frontend - Component

Inside your form component, create a div with an id of `vitnode_recaptcha` to render the captcha.

```tsx {14}
'use client';

import { useSignUpView } from 'vitnode-frontend/hooks/core/sign/up/use-sign-up-view';
import { Form } from 'vitnode-frontend/components/ui/form';
import { CardContent, CardFooter } from 'vitnode-frontend/components/ui/card';

export const FormSignUp = () => {
  const { form, onSubmit, isReady, isSuccess } = useSignUpView();

  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit(onSubmit)}>
        <CardContent className="space-y-4">
          <div id="vitnode_recaptcha" />
        </CardContent>
      </form>
    </Form>
  );
};
```

### Frontend - Hook

Inside your hook, import the `useCaptcha` hook and call the `getTokenFromCaptcha` function to get the captcha token.

```ts
import { useCaptcha } from 'vitnode-frontend/hooks/use-captcha';

export const useSignUpView = () => {
  const { getTokenFromCaptcha, isReady } = useCaptcha();

  const onSubmit = async (values: z.infer<typeof formSchema>) => {
    const token = await getTokenFromCaptcha();
    if (!token) return;

    const mutation = await mutationApi({ ...values, token });
  };

  return { isReady, onSubmit };
};
```

`isReady` is a boolean that indicates if the captcha is ready to be used.

### Frontend - Fetcher

Inside your fetcher, add `x-vitnode-captcha-token` to the headers.

```ts {12-14, 25}
'use server';

import { revalidatePath } from 'next/cache';

import {
  Core_Sessions__Sign_Up,
  Core_Sessions__Sign_UpMutation,
  Core_Sessions__Sign_UpMutationVariables,
} from '@/graphql/graphql';
import { fetcher } from '@/graphql/fetcher';

interface Args extends Core_Sessions__Sign_UpMutationVariables {
  token: string;
}

export const mutationApi = async (variables: Args) => {
  try {
    const { data } = await fetcher<
      Core_Sessions__Sign_UpMutation,
      Core_Sessions__Sign_UpMutationVariables
    >({
      query: Core_Sessions__Sign_Up,
      variables,
      headers: {
        'x-vitnode-captcha-token': variables.token,
      },
    });

    revalidatePath('/', 'layout');

    return { data };
  } catch (error) {
    return { error };
  }
};
```
